# 🧪 SwagLabs Automation Project

## 📌 Overview

UI automation project built with Selenium WebDriver, C#, NUnit, and the Page Object Model (POM) design pattern.

The project automates core user login scenarios on the Swag Labs demo application.

---

## 🛠️ Tech Stack

- C#
- .NET 8
- Selenium WebDriver
- NUnit
- Page Object Model (POM)

---

## 📂 Project Structure

Pages/
- BasePage.cs
- LoginPage.cs
- HiddenMenuPage.cs
Tests/
- BaseTest.cs
- LoginTests.cs
- HiddenMenuPageTests.cs

---

## ✅ Automated Test Scenarios

- Valid login with standard_user
- Invalid login with incorrect credentials
- Logout flow handling (POM expansion in progress)
