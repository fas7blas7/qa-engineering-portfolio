# 🧪 JavaScript Unit Testing

## 📌 Overview

This project contains JavaScript fundamentals practice combined with unit testing using Mocha and Chai.

The goal is to improve problem-solving, coding fundamentals, and test-writing skills through small utility functions and test-driven thinking.

## 🛠️ Tech Stack

* JavaScript
* Node.js
* Mocha
* Chai

## 📂 Project Structure

```text
src/
├── calculator.js
└── stringUtils.js

test/
├── calculator.test.js
└── stringUtils.test.js
```

## ✅ Implemented Functions

### Calculator

* add()
* subtract()
* multiply()
* divide()

### String Utilities

* reverseString()
* isPalindrome()
* countWords()

## 🧪 Test Coverage

The project includes:

* Positive test cases
* Negative test cases
* Edge case validation
* Error handling tests

## ▶️ Run Tests

```bash
npm install
npm test
```

## 🎯 Learning Goals

* JavaScript fundamentals
* Functions and modules
* Array and string manipulation
* Unit testing with Mocha and Chai
* Test design and edge case analysis
